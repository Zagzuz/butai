**Butai**<br>
A theater event aggregator that maps timetables and schedules into a unified interface.
* **Rust Backend:** Manages data orchestration and validation.
* **Worker Layer:** Handles daily automated scraping of theater sources.
* **WASM Frontend:** Fast client-side map rendering and data filtering.
  Built for speed and reliability using a full-stack Rust architecture. Optimized for daily synchronization and high-performance geospatial visualization.
